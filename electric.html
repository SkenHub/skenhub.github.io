電気班

電気班はロボットの回路・通信・ソフトウェアを担当します。安全で再現性の高い動作を実現するため、ハードとファームを一体で設計します。

扱う技術

  • マイコン (STM32) / ROS 2
  • モータードライバ / モジュール回路
  • CAN / SPI / UART / 無線通信
    • etc.

ソフト & ツール

  • System Workbench for STM32
  • ROS 2
  • KiCad
  • GitHub
