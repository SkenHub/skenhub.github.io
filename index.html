私たちについて

舞鶴高専 創造技術研究会（S-ken）は、高専ロボコン全国大会 優勝と「ものづくりで仲間を笑顔にすること」を目標に活動する学生ロボット開発チームです。
設計・加工・回路・制御・ソフトウェアまで一気通貫で自分たちの手で行い、創造力 / 技術力 / チーム力を高めながら競技に挑戦しています。

平日放課後と休日に部室・実習工場で活動しています。失敗を歓迎し、試作を高速に繰り返す「Try Fast」文化が根付いています。

班紹介

創造技術研究会は 機械班 と 電気班 の2班が密に連携してロボットを完成させます。要件定義フェーズでは両班合同で競技分析とアイデア出しを行い、開発中はインターフェース（取り付け寸法・電源仕様・通信プロトコル）を週次レビューで同期します。

班をまたいだ兼任も可能で、CAD設計しながらファームを書いたり、回路を作りながら加工を手伝うメンバーもいます。

機械班

機械班はロボットの構造・駆動・エンドエフェクタを設計・製作するチームです。要件から荷重/強度計算を行い、軽量性・剛性・整備性のバランスを最適化します。

  • 使用ツール: 3D CAD (Fusion 360), 2D CAD, 3Dプリンタ, レーザー加工機, 旋盤, フライス盤
  • 主なタスク: アイデア試作 / 駆動方式検討 / 設計レビュー / 部品加工 / 組立 / メンテ / 改良
  • 今年の焦点: モジュール化と軽量化（部品点数 15% 削減目標）

電気班

電気班はロボットの電源・駆動制御・センシング・通信・ソフトウェアを担当します。安全で再現性の高い動作を実現するため、ハードとファームを一体で設計します。

  • 扱う技術: マイコン(Cortex-M), モータードライバ, 電源レギュレーション, I2C/SPI/UART, 無線通信
  • ソフト: 組込みC/C++ / PlatformIO / Pythonログ解析 / GitHub Actionsでビルド確認
  • 開発フロー: 要件→回路試作→基板設計→ファーム実装→動作ログ解析→最適化
  • 今年の焦点: センサ融合 + フィードフォワード制御 / 回路モジュール共通化
